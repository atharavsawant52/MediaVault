# MediaVault

MediaVault is a responsive media search web app built with **React** and **Redux Toolkit**.  
It allows users to search, save, and download **images, videos, and GIFs** using public APIs with infinite scrolling and optimized performance.

## Features
- Search images, videos & GIFs
- Infinite scroll (Intersection Observer)
- Save to collection (Redux + LocalStorage)
- Download media using Blob API
- Fully responsive design

## Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS
- Unsplash, Pexels & Klipy APIs

## Setup
```bash
npm install
npm run dev
